## 폴더 역할

### `C:\dev\projects\Tilon`

GitHub 원격 저장소와 연결된 실제 프로젝트 루트다.

원격:

```text
https://github.com/Dodi029/Tilon.git
```

현재 `notion_pdf` 프로젝트는 이 저장소 안의 하위 프로젝트다.

### `C:\dev\projects\Tilon\notion_pdf`

현재 작업 중인 Notion PDF 변환기 프로젝트다.

주요 역할:

- Flask 로컬 서버
- Playwright 기반 HTML/URL 렌더링
- PNG 기반 1페이지 PDF 생성
- 테스트 및 문서 관리

### `C:\dev\projects\Tilon\prompts`

Codex 지시문과 작업 요청 템플릿을 보관하는 폴더다.

현재 작업에서 참고한 공통 지시문:

```text
C:\dev\projects\Tilon\prompts\codex-기본지시문.md
```

### `C:\dev\projects\Tilon\docs`

저장소 전체 문서 폴더다.

프로젝트별 문서는 현재 `notion_pdf\docs` 안에도 별도로 존재한다.

### `C:\dev\projects\Tilon\scripts`

저장소에 포함할 수 있는 공통 스크립트 위치다.

### `C:\dev\projects\Tilon\deploy`

배포 관련 파일을 둘 수 있는 위치다.

### `C:\dev\projects\Tilon\tools`

개발 보조 도구를 둘 수 있는 위치다.

### `C:\dev\logs`

로컬 실행 로그나 임시 기록을 저장하는 위치다. Git에 직접 올릴 필요가 없는 로그에 사용한다.

### `C:\dev\sandbox`

임시 실험, 일회성 파일, 검증용 산출물을 둘 수 있는 위치다.

### `C:\dev\scripts`

PC 전체에서 공통으로 사용할 개인 로컬 스크립트 위치다.

### `C:\dev\vm`

VM, 이미지, ISO, VHDX, qcow2 같은 대용량/로컬 전용 파일을 두는 위치다.

## 다른 PC에서 동일 환경 복원

Windows 기준:

```powershell
mkdir C:\dev
mkdir C:\dev\projects
cd C:\dev\projects
git clone https://github.com/Dodi029/Tilon.git
cd C:\dev\projects\Tilon\notion_pdf
python -m pip install -r requirements.txt
python -m playwright install chromium
python -m unittest discover -s tests
python tests\run_server_pdf_flow.py
python app.py
```

Mac/Linux 기준:

```bash
mkdir -p ~/dev/projects
cd ~/dev/projects
git clone https://github.com/Dodi029/Tilon.git
cd Tilon/notion_pdf
python -m pip install -r requirements.txt
python -m playwright install chromium
python -m unittest discover -s tests
python tests/run_server_pdf_flow.py
python app.py
```

Windows 경로를 전제로 한 문서와 테스트 출력이 일부 있으므로, Mac/Linux에서 사용할 때는 경로 표기만 OS에 맞게 바꾸면 된다.
