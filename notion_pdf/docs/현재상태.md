## 20. Cloudflare Tunnel 공개 방식

현재 개발/테스트 공개 구조는 다음과 같다.

```text
Mac mini
→ Flask 서버 실행
→ Cloudflare Tunnel 연결
→ 외부 HTTPS 주소 생성
→ 다른 PC/사용자 접근 가능
```

현재 프로젝트 기준 서버 실행 위치:

```bash
cd ~/git/dev/projects/Tilon/notion_pdf
source .venv/bin/activate
python app.py
```

기본 로컬 접속 주소:

```text
http://127.0.0.1:5000
```

Cloudflare Tunnel 실행:

```bash
cloudflared tunnel --url http://127.0.0.1:5000
```

성공하면 다음 형식의 임시 외부 HTTPS 주소가 표시된다.

```text
https://xxxxx.trycloudflare.com
```

이 주소를 다른 PC, Windows 노트북, 외부 사용자에게 전달하면 현재 Mac 개발 서버에 접근할 수 있다.

주의사항:

- Flask 서버 터미널이 종료되면 외부 접속이 불가능하다.
- `cloudflared` 터미널이 종료되면 외부 접속이 불가능하다.
- `trycloudflare.com` 주소는 임시 주소이며 재실행 시 바뀔 수 있다.
- Mac 절전 모드 진입 시 연결이 끊길 수 있다.
- 현재 구조는 개발/테스트 목적이다.
- OCR 및 고화질 PDF 생성은 CPU/RAM 사용량이 크다.

상세 문서는 `docs/배포가이드.md`에 기록했다.
