- 캡처 이미지 높이가 긴 콘텐츠를 반영해 `9000px`보다 큰지 확인했다.
- PDF 페이지 높이가 폭보다 큰지 확인했다.
- PDF 페이지 크기 비율이 PNG 크기 비율과 일치하는지 확인했다.

### 서버 플로우 테스트

명령:

```bash
python tests\run_server_pdf_flow.py
```

확인 내용:

- 테스트 스크립트가 `python app.py`로 Flask 서버를 실행한다.
- `http://127.0.0.1:5055` 접속 가능 여부를 확인한다.
- 샘플 HTML 파일을 `/convert/file`로 업로드한다.
- `/job/<job_id>`를 polling해 변환 완료를 기다린다.
- `/download/<job_id>`로 PDF를 다운로드한다.
- `pypdf`로 PDF가 정확히 1페이지인지 확인한다.
- PDF 페이지 높이가 폭보다 커서 긴 콘텐츠가 한 페이지에 반영됐는지 확인한다.
